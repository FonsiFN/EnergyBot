Tu Bot de Discord

EnergyBot está diseñado para ayudarte a gestionar tu servidor de Discord de manera eficiente y divertida.

Gestión de Servidores

EnergyBot permite una moderación eficaz de tu servidor, garantizando un ambiente amigable para todos los miembros.

Interacción con Usuarios

Facilita la interacción a través de encuestas, juegos y otros comandos que mejoran la participación de la comunidad.

Automatización de Tareas

Configura comandos automáticos para recordatorios, anuncios y otras tareas que mejoran la experiencia en tu servidor.

Comandos Principales

/help - Muestra todos los comandos disponibles.

/ping - Comprueba la latencia del bot.

/info - Muestra información sobre el bot y sus características.

/invite - Envía el enlace de invitación del bot a otros servidores.

/roll - Lanza un dado virtual y devuelve un número aleatorio.

Preguntas Frecuentes

¿Cómo puedo invitar a EnergyBot a mi servidor?

Utiliza el comando /invite para obtener el enlace de invitación y seguir las instrucciones.

¿Qué tipos de comandos están disponibles?

EnergyBot ofrece comandos para moderación, información, entretenimiento y mucho más. Usa /help para ver la lista completa.

¿Puedo personalizar la configuración del bot?

Sí, puedes ajustar la configuración del bot a través de comandos específicos que se encuentran en la sección de comandos.

¿Hay algún costo para usar EnergyBot?

No, EnergyBot es completamente gratuito. Sin embargo, algunas características avanzadas pueden requerir donaciones para mantener el servicio.
